# 238. 除自身以外数组的乘积
```
class Solution {
    public int[] productExceptSelf(int[] nums) {
        int n = nums.length;
        int[] p = new int[n];
        for (int i = 0; i < n; i++) {
            p[i] = 1;
        }
        for (int i = 1; i < n; i++) {
            p[i] = p[i - 1] * nums[i - 1];
        }
        int s = 1;
        for (int i = n - 1; i >= 0; i--) {
            p[i] *= s;
            s *= nums[i];
        }
        return p;
    }
}
```
