# 22.括号生成

```
class Solution {
    List<String> ans = new ArrayList<>();

    public List<String> generateParenthesis(int n) {
        dfs(n, 0, 0, "");
        return ans;
    }

    private void dfs(int n, int lc, int rc, String seq) {
        if (lc == n && rc == n) {
            ans.add(seq);
            return;
        }
        if (lc < n) {
            dfs(n, lc + 1, rc, seq + "(");
        }
        if (rc < n && lc > rc) {
            dfs(n, lc, rc + 1, seq + ")");
        }
    }
}
```
