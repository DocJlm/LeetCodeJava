# 29.两数相除
```
class Solution {
    public int divide(int x, int y) {
        long[] exp = new long[33]; 
        int expSize = 0;
        boolean isMinus = false;
        
        if ((x < 0 && y > 0) || (x > 0 && y < 0)) {
            isMinus = true;
        }
        long a = Math.abs((long) x);
        long b = Math.abs((long) y);
        for (long i = b; i <= a; i = i + i) {
            exp[expSize++] = i;
        }
        long res = 0;
        for (int i = expSize - 1; i >= 0; i--) {
            if (a >= exp[i]) {
                a -= exp[i];
                res += 1L << i;
            }
        }

        if (isMinus) {
            res = -res;
        }

        if (res > Integer.MAX_VALUE || res < Integer.MIN_VALUE) {
            return Integer.MAX_VALUE;
        }
        return (int) res;
    }
}
```
