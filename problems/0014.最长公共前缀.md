# 14.最长公共前缀
```java
class Solution {
    public String longestCommonPrefix(String[] strs) {
         String res = "";
        if (strs.length == 0) return res;

        for (int i = 0; ; i++) {
            if (i >= strs[0].length()) return res;
            char c = strs[0].charAt(i);  

            for (String str : strs) {
                if (str.length() <= i || str.charAt(i) != c) {
                    return res;
                }
            }
            res += c;
        }
    }
}
```
